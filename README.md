# Remove Background PCF Component for Power Apps

A Power Apps Component Framework (PCF) control that integrates with Remove.bg API to automatically remove backgrounds from images. Perfect for Power Apps Canvas and Model-Driven apps.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Power Platform](https://img.shields.io/badge/Power%20Platform-PCF-orange.svg)

## 🌟 Features

- ✨ **One-Click Background Removal** - Upload image and remove background with a single click
- 🎨 **Fully Customizable UI** - Customize colors, fonts, and button texts
- 📥 **Download & Copy** - Download result or copy to clipboard
- 🖼️ **Custom Placeholder** - Support custom placeholder images
- 📱 **Responsive Design** - Works on desktop and mobile devices
- 🔒 **Secure API Integration** - Built-in Remove.bg API integration
- ⚡ **Real-time Processing** - See results instantly
- 🎯 **Progress Indicators** - Visual feedback during processing

## 🛠️ Prerequisites

- **Node.js** (v14.x or higher)
- **.NET SDK** (6.0 or higher)
- **Power Platform CLI** (`pac`)
- **MSBuild Tools** (Visual Studio 2019/2022 or Build Tools)
- **Remove.bg API Key** - [Get Free API Key](https://www.remove.bg/users/sign_up)
